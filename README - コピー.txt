MSX テープソフトを実行（暫定版 v0.20）

<環境>
MSX2以降 VRAM128k以上 MSX-DOSが起動すること

<使い方>
1.以下ファイルを入れたMSX-DOS起動FDを用意
・RUN.BAT ・・・ 起動用 BATファイルのサンプル
・BLOADCAS.COM ・・・ *.CAS ファイルをVRAMへ読み込む DOSコマンド
・RUNBAS.ASC ・・・ BASIC 起動後にマシンを自動実行する BASICプログラム
・実行したい CASファイル

2.MSX-DOS が起動したら、起動用 BATファイルを実行

（補足）起動用 BATファイルは以下の内容
BLOADCAS <ファイル名.CAS>
BASIC RUNBAS.ASC

<対象>
・BSAVE"CAS:" されたマシン後プログラムファイル（*.CAS）
・プログラム実行後にてテープを R/Wするものは対象外
（"CAS:"デバイスを再現したものではないため）

<お断り>
・実験的にプログラムを組んでいるため、動作保証は出来かねます。
・所持していないため、色々なテープソフトので動作検証ができていません。
・起動しない、途中で止まるなどが起こる可能性があります。
・予告なくプログラムの修正や削除、公開中止をすることがあります。
